# Additional features

The Oracle module was designed to be as flexible and customizable as possible, and as a result, much of the high-level customization is left up to the developer. Things rewards, slashing and more sophisticated ways of determining concensus can all be implemented outside of the Oracle module.

If you're interested in developing building any of these features, we suggest taking a look at the Terra Oracle Module (opens new window)
